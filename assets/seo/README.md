# SEO Social Preview Images

This folder is reserved for page share-preview images (Open Graph / Twitter images).

## Purpose
These images are used when Musicatea pages are shared on WhatsApp, Facebook, X/Twitter, Telegram, Slack, Discord, and similar platforms.

## Upload rule
When you prepare the final images, upload them into this folder using the exact filenames below.

## Recommended format
- File type: JPG
- Aspect ratio: 1200x630
- Keep text large and readable
- Use high contrast
- Keep page title short
- Avoid putting too much text on the image
